This node run the waiterBot Demo The robot moves to its target location, stoping every time there is an obstical, and it retruns to the food/drink pick up location only once it has no more food It publishes geometry_msgs::Twist messages from the "/mobile_base/commands/velocity" topic It subscribes to the the "/scan", "force", "/odom" topic.
